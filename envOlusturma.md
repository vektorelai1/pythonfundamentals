# Windows Kullanıcısı
## Oluşturmak için
* python -m venv [ortamismi] => python -m venv env 
## Ortamı Aktif Hale Getirmek için
* env\Scripts\activate
## Pasif Hale Getirmek için
* env\Scripts\deactivate

# Mac&Linux Kullanıcısı
* python3 -m venv env
## Ortamı Aktif Hale Getirmek için
* source env\bin\activate
